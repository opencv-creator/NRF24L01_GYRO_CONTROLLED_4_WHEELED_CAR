# NRF24L01_GYRO_CONTROLLED_4_WHEELED_CAR
NRF24L01_GYRO_CONTROLLED_4_WHEELED_CAR

This project has NRF24L01 trasmitter connected to Arduino Nano and  NRF24L01 reciever connected to Arduino UNO

IMPORTANT POINTS:

to make this project plese purchase 
NRF24L01 socket board present in below link
https://robu.in/product/3-3v-adapter-board-for-24l01-wireless-module/

purchase NRF24L01 with Antenna (this is directional antenna,so this antenna doesnt work properly when the antennas are not in same direction and same angle.
https://robu.in/product/2-4ghz-nrf24l01palna-sma-antenna-wireless-transceiver-communication-module-1km/

to make NRF24L01 with Antenna to work , please place a 10 micro Farad or 100 micro farad capacitor near the vcc and gnd pin of the NRF24L01.

NRF24L01 works with 3.3v
NRF24L01 when connected to 8 pin socket board can work with 5v.

seperate good power supply has to be used for the NRF24L01 to work properly.

use the democode present for NRF24L01 Transmitter and Receiver present in this page to check the functionality of the NRF24L01.
https://forum.arduino.cc/t/simple-nrf24l01-2-4ghz-transceiver-demo/405123
